LibNu is a Python ctypes wrapper for the [Nu][] library.

[![](https://travis-ci.org/eliteraspberries/python-libnu.svg)][build-status]


## Requirements

LibNu requires Python and the [Nu][] library.

To run unit tests, [pytest][] is required.
