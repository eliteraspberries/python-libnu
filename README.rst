LibNu is a Python ctypes wrapper for the
`Nu <http://github.com/eliteraspberries/nu>`__ library.

|image0|

Requirements
------------

LibNu requires Python and the
`Nu <http://github.com/eliteraspberries/nu>`__ library.

To run unit tests, `pytest <https://pytest.org/>`__ is required.
